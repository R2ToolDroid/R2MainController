# R2MainController
Full Droid Control System with 3 or 4 input channels.
Communication via UART Serial

Control your Droid from differend Systems.

I use a Arduino MEGA 2560


---Input---

1. R2Touch App
2. RCInputController
3. CoinButton
4. couple of automatic sensors


---Output---

1. DomeController
2. Marcduino Boards
3. BodyGadgetController
